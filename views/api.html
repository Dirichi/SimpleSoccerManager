SOCCER MULTIVERSE

HOME ABOUT API

This API provides free access to data about all the games played on the soccer multiverse. The major end point is

https://simplesoccermanagers.herokuapp.com/api

ALL GAMES

Data about all games can be accessed from the url

/api/allgames

Note that the data returned contains information about the status about a game at ten minute intervals, ie at time 0, 10, 20, 30, etc. A useful unique identifier for aggregating data pertaining to the same game accross different times, is the key attribute

A SPECIFIC GAME

Data about a single game

/api/allgames/gameKey/GAME_KEY

where GAME_KEY referes to the value of that game's key attribute. Note that the data returned contains information about the status of said game at ten minute intervals, ie at time 0, 10, 20, 30, etc.

FINIHED GAMES

Data about all completed games can be accessed from the url

/api/allgames/ended/

GAMES WON BY TEAM X

Data about finished games won by a particular team can be accessed from the url

/api/allgames/wonBy/TEAM_CODE

Since there are currently only two teams in soccer multiverse, the TEAM_CODE parameter can be either "A" or "B"
